# Real-Time Dashboard with websocket and PostgresQL Listen/Notify


### Tech Stack
* Go
* Websocket
* jQuery
* PostgresQL Listen/Notify
* Bootstrap


Build linux binary
    cd src
    CGO_ENABLED=0 GOOS=linux go build -o ../cmd/websocket-dashboard-linux

Deploy with docker-compose
    ./deploy-docker.sh
